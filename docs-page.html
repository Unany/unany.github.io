Introduction Last updated: 2019-06-01

Overview

The Ultimate Grids Engine it’s a framework for grid-based games, which allows you to create grid-based games quickly and effortless.

Among other things it provides you with: entities system, grid based movement, range and path finding algorithms, full level editor which allows you to populate grids with tiles and entities, modular attack system, interactive objects, packed with games and mechanics examples, and a lot more.

Fully supports Rectangle, Hexagon and Isometric grids for both 2D and 3D games.

Importing

  1. Create a new project from Unity Hub, pick the latest stable version of Unity as your Unity version, and 2D or 3D as the Template.
  2. Go to the Asset Store window and import the asset (it has to be in an empty project, not an existing one).
  3. You will be warned that importing the asset will overwrite your current project settings (that’s normal), click on Import.
  4. Once this is complete, you’ll most likely get a prompt saying “This Unity Package has Package Manager dependencies”. This is also normal, click on “Install/Upgrade”.
  5. After that you’ll get a list of the contents of the engine, don’t touch anything, and click on Import in the bottom right corner.
  6. Import will take a while. Once import is complete, a welcome window should prompt (Fig. bellow) if it shows any dependencies are missing please click on install for each one and you’re ready to use the engine. Have fun!
  7. In the case you're getting errors on a fresh install follow this last step, else ignore it.
    • Go to the Window -> Package Manager.
    • Find and install these packages: 2D Tilemap Editor, Post Processing,CineMachine and TextMesh Pro.

Quick Start

The Ultimate Grids Engine is your ultimate solution for creating ANY GRID-Based game in Unity.

It allows you to create and populate all kinds of regular and semi-regular grids, including rectangular, hexagonal and isometric grids quickly and effortless. Includes a robust grid entities and tiles system, advanced grid-movement, A* path finding, character controllers, interactive objects and everything you need to create your own grid-based game.

Anything is possible, from tactics games like Into the Breach and Advanced Wars, to puzzle games like Monument Valley and Tetris, through roguelikes/lites such as Crypt of the NecroDancer, verby nouns like Crossy Road or even match 3 like Candy Crush, and really any game where you need a Grid of any kind and Entities to move on it.

It is the best framework for grid based games out there, whether you're new to Unity, or an experienced developer. Easy to use, packed with features, built with good practices at its core, and constantly updated, it's the ultimate tool to get yourself a head start and have fun creating your game.

Time Manager

The Time Manager is the component that handles the time for the current game as well as setting the blend between night and day skyboxes. When creating a new scene using the Weather Magic Menu, this component is attached to an object called ‘GameManager’. The Game Manager component is seperated into three parts: The time, season and celestial manager.

The Time Manager Component:

Properties

Seconds In A Day The amount of real world seconds for a day/night cycle to go from 00:00:00 to 24:00:00
Time Multiplier Adjusts the rate at which time passes
Time Curve The current season assets time curve
Evaluated Value The evaluated value to lerp the skyboxes from the current time curve
DD:HH:MM:SS The current ingame time, in an standard time format
On Hour Triggers an event everytime the hour value is incremented
On Day Triggers an event everytime the day value is incremented
On Day Asset Triggers an event whenever the time curves second key is
On Night Asset Triggers an event whenever the time curves fourth key is

API

Seconds In A Day The amount of real world seconds for a day/night cycle to go from 00:00:00 to 24:00:00
Time Multiplier Adjusts the rate at which time passes
Time Curve The current season assets time curve
Evaluated Value The evaluated value to lerp the skyboxes from the current time curve

Tip

In order to speed up the day/night cycle, if for example your character is sleeping, speed up the time multiplier in order to see a rapidly changing skybox

Weather Scriptable Object

The Time Manager is the component that handles the time for the current game as well as setting the blend between night and day skyboxes. When creating a new scene using the Weather Magic Menu, this component is attached to an object called ‘GameManager’. The Game Manager component is seperated into three parts: The time, season and celestial manager.

The Weather Scriptable Object:

Properties

Weather Name The name of this weather asset
Weather Weighting The chance for this weather asset to occur in the season. Every additional '1' increases the chance by 1 for that weather asset.
Skybox Tint The background tint of the skybox
Unique Skybox Allows you to select a new skybox asset for this weather theme
Image The skybox image for this weather asset
Weather Pool Object To Pool The prefab to pool
Amount To Pool How many objects of the prefab do you want to instantiate. This is not at once, but rather the total, which can be pooled from
Chance To Change What is the chance for the prefab to occur
Next To Camera Will the object be placed next to the camera (i.e. at the cameras transform)
Offset Amount The amount to offset the prefab in relation to the main camera
Connect To Vegetation Shader Enable to allow access to third party shaders

Tip

In order to speed up the day/night cycle, if for example your character is sleeping, speed up the time multiplier in order to see a rapidly changing skybox
